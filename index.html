Bubble Sort

1. compare two adjacent elements, swap the bigger one to right


2. repeat step 1, until the last element is the biggest.


3. repeat step 1 + step2, the original largest element which already in the last cannot be changed.


4. repeat the above step, until do not need to compare any more.


Bogo sort

1. Check the sorting of elements, wrong or correct?


2. If wrong sorting, randomly generate a new sequence, repeat step 1.


3. If correct, complete sorting


Insertion sort

1. Before sorting the element at index N, make sure from first element to N-1 element already sorted.


2. Then compare N with the previous sorted sequence.


3. And so on, until complete sorting


image source: http://java-latte.blogspot.com/2013/06/insertion-sort-algorithm.html


Selection sort

1. Prepare a new list.


2. Find the smallest element in the original sequence and assign to the last position in the new list.


3. Repeat step 2, until complete sorting


Quick sort

1. Choose any one number in the sequence as the pivot.


2. The original sequence is divided into two new sequences.


3. The elements in one sequence are all less than pivot, and the elements in other one sequence are all larger than pivot.


4. These two new sequences repeat the step 1 to step 3, until it can not be divided into a new sequence.


5. Individual items are already sorted.


6. Combine the individual items as a sorted sequence.
